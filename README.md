
# MontinkSneakers 🧢👟

Este é o projeto desenvolvido para o teste técnico da empresa **Montink**, onde foi solicitado a construção de uma página de produto de E-commerce utilizando **React** e **Tailwind CSS**. O projeto foi desenvolvido com **Vite**, animado com **Framer Motion** e está hospedado na **Vercel**.

## 🔗 Acesse o projeto online

👉 [https://vercel.com/felipe-duartes-projects-a27faf87/montink-ecommerce](https://vercel.com/felipe-duartes-projects-a27faf87/montink-ecommerce)

---

## 🛠️ Tecnologias Utilizadas

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Vercel](https://vercel.com/) (Deploy)

---

## 📦 Como rodar localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/montink-ecommerce.git
   cd montink-ecommerce
   ```

2. Instale as dependências:
   ```bash
   yarn
   ```

3. Rode o projeto:
   ```bash
   yarn dev
   ```

## ✅ Funcionalidades Implementadas

- ✅ Imagem principal do produto (35% da tela)
- ✅ Miniaturas clicáveis que trocam a imagem principal
- ✅ Título e preço dinâmicos
- ✅ Seletores dinâmicos de variantes (Tamanho e Cor)
- ✅ Campo de CEP com verificação via [ViaCEP](https://viacep.com.br/)
- ✅ Exibição do endereço completo ao inserir um CEP válido
- ✅ Persistência dos dados preenchidos por 15 minutos (mesmo após recarregar a página)

---

## 💡 Observações

- Todos os dados foram estruturados dinamicamente, garantindo flexibilidade para mudanças futuras.
- O design foi inspirado em grandes e-commerces como Mercado Livre, Shopee e Amazon, buscando uma experiência familiar e fluida para o usuário.
